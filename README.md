# 🌹 Mother's Day Website

A beautiful, interactive Mother's Day website created with love for your mom!

## Features

✨ **Segmented Navigation** - Navigate through 5 different sections:
1. **Title Section** - Beautiful intro with flowers
2. **Message Section** - Heartfelt message for your mom
3. **Photo Gallery** - Display your cherished memories
4. **Music Section** - Play a special song
5. **Quote Section** - Inspiring Mother's Day quote

🎨 **Beautiful Design**
- Gradient pink and purple theme
- Smooth animations and transitions
- Fully responsive (works on mobile and desktop)
- Progress bar showing your progress
- Keyboard navigation (arrow keys work too!)

## How to Use

### 1. Add Your Photos

Replace the placeholder images with your own photos:

1. Upload your photos to the repository:
   - `photo1.jpg` (for image 1)
   - `photo2.jpg` (for image 2)
   - `photo3.jpg` (for image 3)

2. In `index.html`, find the gallery section and update these lines:
   ```html
   <img src="photo1.jpg" alt="Beautiful moment with mom">
   <img src="photo2.jpg" alt="Special memory">
   <img src="photo3.jpg" alt="Precious moments">
   ```

### 2. Add Your Song

1. Upload your song file to the repository as `mothers-day-song.mp3`

2. Make sure the audio file format is MP3 or adjust the source type in the code if needed

### 3. Customize the Message

Edit the message in Segment 2 to add your personal touch:
- Change the greeting
- Add specific memories
- Personalize the closing

### 4. View Your Website

Enable GitHub Pages:
1. Go to your repository Settings
2. Scroll to "GitHub Pages" section
3. Select the `main` branch as source
4. Your site will be available at: `https://madiwalegargi05-droid.github.io/Mothers-Day/`

## Navigation

- **Next Button** - Move to the next section
- **Previous Button** - Go back to previous section
- **Keyboard** - Use arrow keys (← →) to navigate
- **Progress Bar** - Visual indicator of your progress through the site

## Tips

💡 **Photo Requirements:**
- Recommended size: 300x300px or larger
- Format: JPG, PNG, or WebP
- Multiple photos work best for a nice gallery

🎵 **Song Requirements:**
- Format: MP3 (best browser support)
- Any audio file can work (WAV, OGG, etc.)
- Keep file size reasonable for faster loading

🎨 **Customization:**
Feel free to edit the colors, fonts, and messages to make it more personal!
